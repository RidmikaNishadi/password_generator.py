A simple Python project that generates secure random passwords based on user input.
Features
- Generates random passwords
- User can choose password length
- Uses letters, numbers, and symbols
- Every password is unique

The program uses Python's random module to select random characters from a pool of:
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

It builds the password one character at a time using a loop.
